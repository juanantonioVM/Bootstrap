Grid en CSS

Ejemplo Descripción 
.container { display: grid; }
 Establece un contenedor como un grid container. 
.container { grid-template-columns: repeat(3, 1fr); }
 Crea tres columnas del mismo tamaño en el grid. 
.item { grid-column: span 2; }
 Un elemento puede ocupar dos columnas en el grid. 
.container { gap: 10px; }
 Establece un espacio de 10 píxeles entre las filas y columnas del grid. 
.container { grid-template-rows: 100px auto; }
 Crea dos filas, la primera de 100 píxeles y la segunda automática. 
.item { grid-area: 1 / 1 / 2 / 3; }
 Ubica un elemento en la primera fila y ocupa las dos primeras columnas del grid. 
.container { grid-template-areas: "header header" "main aside"; }
 Define áreas en el grid, donde 'header' ocupa las dos primeras columnas en la primera fila.
