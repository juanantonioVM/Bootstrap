Transiciones en CSS

Ejemplo Descripción 
div { transition: all 0.3s ease; }
 Aplica una transición a todas las propiedades que cambien en un tiempo de 0.3 segundos. 
div:hover { background-color: blue; }
 Cambia el color de fondo a azul cuando el ratón pasa sobre el div. 
div { width: 100px; }
 Establece un ancho inicial de 100 píxeles para el div. 
div:hover { width: 200px; }
 Aumenta el ancho del div a 200 píxeles al pasar el ratón. 
div { border: 2px solid black; }
 Establece un borde negro de 2 píxeles en el div. 
div:hover { border: 5px solid red; }
 Cambia el borde a rojo y a 5 píxeles cuando se pasa el ratón. 
button { transition: background-color 0.5s; }
 Aplica una transición de 0.5 segundos al color de fondo de un botón.
