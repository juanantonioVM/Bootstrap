Espaciado en CSS

Ejemplo Descripción 
p { margin: 20px; }
 Establece un margen de 20 píxeles alrededor de todos los párrafos. 
div { padding: 15px; }
 Aplica un relleno de 15 píxeles dentro de todos los div. 
h1 { margin-top: 50px; }
 Establece un margen superior de 50 píxeles para todos los encabezados <h1>. 
ul { padding-left: 30px; }
 Aplica un relleno a la izquierda de 30 píxeles en todas las listas no ordenadas. 
section { margin-bottom: 40px; }
 Establece un margen inferior de 40 píxeles para todas las secciones. 
h2 { padding: 10px 5px; }
 Aplica un relleno de 10 píxeles arriba y abajo, y 5 píxeles a la izquierda y derecha a todos los encabezados <h2>. 
li { margin: 5px 10px; }
 Establece un margen de 5 píxeles arriba y abajo, y 10 píxeles a los lados para todos los elementos de lista.
