Bordes en CSS

CSS3 permite crear bordes redondeados con la propiedad border-radius, lo que facilita dar un aspecto más moderno a los elementos.

Ejemplo Descripción 
-moz-border-radius: 10px;
-webkit-border-radius: 10px;
border-radius: 10px;
Aplica un borde redondeado de 10 píxeles en todas las esquinas de la caja, compatible con navegadores como Firefox y Chrome. 
border-radius: 20px 10px;
Redondea solo las esquinas opuestas de un elemento, donde 20px se aplica a la esquina superior izquierda e inferior derecha, y 10px a las otras dos. 
border-radius: 15px 30px 20px 5px;
Redondea cada esquina de la caja de forma personalizada en el orden: superior izquierda, superior derecha, inferior derecha, e inferior izquierda. 
border-radius: 20px / 10px;
Usa valores separados por una barra para crear bordes elípticos, donde el valor a la izquierda indica el radio horizontal y el de la derecha el vertical. 
div { border: 2px solid red; }
 Establece un borde rojo de 2 píxeles alrededor de todos los div. 
p { border-bottom: 1px dashed black; }
 Aplica un borde inferior negro y punteado a todos los párrafos. 
h1 { border: 5px double blue; }
 Establece un borde azul de 5 píxeles y doble alrededor de todos los encabezados <h1>. 
img { border-radius: 50%; }
 Aplica bordes redondeados al img, formando un círculo para imágenes cuadradas. 
section { border-style: dotted dashed solid double; }
 Aplica diferentes estilos de borde a cada lado: puntos, guiones, sólido y doble.
