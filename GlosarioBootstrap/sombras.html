Sombras en CSS

La propiedad box-shadow permite crear sombras en los elementos, ajustando desplazamiento, difuminación y color.

Ejemplo Descripción 
box-shadow: 5px 5px 5px #888;
Crea una sombra desplazada 5px a la derecha y abajo, con 5px de difuminado y color gris. 
box-shadow: -5px -5px 10px rgba(0, 0, 0, 0.5);
Sombra desplazada 5px hacia la izquierda y arriba, con 10px de difuminado y semitransparente. 
box-shadow: 3px 3px 3px #aaa inset;
Crea una sombra interna dentro del elemento, ideal para darle un efecto de profundidad. 
text-shadow: 2px 2px #ff0000;
Aplica una sombra roja al texto, desplazada 2px a la derecha y abajo. 
div { box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5); }
 Aplica una sombra de 5 píxeles a la derecha y abajo, con un difuminado de 10 píxeles. 
h1 { text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.7); }
 Agrega una sombra de texto negra a los encabezados <h1>. 
button { box-shadow: inset 0 0 10px rgba(0, 0, 0, 0.5); }
 Aplica una sombra interior a los botones, dando un efecto de profundidad. 
p { text-shadow: 0px 0px 3px blue; }
 Aplica una sombra azul suave alrededor del texto de todos los párrafos.
