Diseño de Caja en CSS

Ejemplo Descripción 
div { width: 300px; height: 150px; }
 Establece un ancho de 300 píxeles y una altura de 150 píxeles para todos los div. 
div { padding: 20px; }
 Aplica un relleno de 20 píxeles alrededor del contenido del div. 
div { border: 5px solid black; }
 Agrega un borde negro de 5 píxeles a todos los div. 
div { margin: 30px; }
 Establece un margen de 30 píxeles alrededor de todos los div. 
div { box-sizing: border-box; }
 Incluye el padding y el border en el ancho y alto totales del div. 
div { border-radius: 10px; }
 Aplica un radio de borde de 10 píxeles, redondeando las esquinas del div. 
div { background-color: lightgray; }
 Aplica un fondo gris claro a todos los div.
