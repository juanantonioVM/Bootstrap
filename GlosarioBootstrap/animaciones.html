Animaciones en CSS

Ejemplo Descripción 
@keyframes nombreAnimacion {}
 Define una animación con un nombre específico. 
@keyframes nombreAnimacion { from { opacity: 0; } to { opacity: 1; } }
 Ejemplo de animación que hace que un elemento se desvanezca desde invisible a visible. 
div { animation: nombreAnimacion 2s ease-in; }
 Aplica la animación definida a un div durante 2 segundos. 
div { animation: nombreAnimacion 2s infinite; }
 Repite la animación indefinidamente. 
div { animation-delay: 1s; }
 Retrasa el inicio de la animación por 1 segundo. 
div { animation-direction: alternate; }
 Reversa la dirección de la animación en cada ciclo. 
div { animation-fill-mode: forwards; }
 Mantiene el estado final de la animación al terminar.
