Fuentes y Tipografía en CSS

Ejemplo Descripción 
body { font-family: Arial, sans-serif; }
 Aplica la fuente Arial (o una fuente sans-serif alternativa) al texto del cuerpo. 
h1 { font-size: 24px; }
 Establece el tamaño de la fuente a 24 píxeles para todos los encabezados <h1>. 
p { font-weight: bold; }
 Aplica negrita al texto de todos los párrafos. 
h2 { font-style: italic; }
 Aplica cursiva al texto de todos los encabezados <h2>. 
h1 { text-align: center; }
 Centra el texto de todos los encabezados <h1>. 
li { letter-spacing: 2px; }
 Aplica un espaciado de 2 píxeles entre las letras de todos los elementos de lista. 
blockquote { font-size: 18px; line-height: 1.5; }
 Establece un tamaño de fuente de 18 píxeles y una altura de línea de 1.5 para citas en bloque.
