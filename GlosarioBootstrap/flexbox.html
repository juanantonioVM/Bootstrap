Flexbox en CSS

Ejemplo Descripción 
.container { display: flex; }
 Establece un contenedor como un flex container. 
.item { flex: 1; }
 Permite que todos los elementos flexibles dentro del contenedor crezcan por igual. 
.container { justify-content: center; }
 Centra horizontalmente los elementos dentro del contenedor flex. 
.container { align-items: stretch; }
 Establece que los elementos flexibles estiren para llenar el contenedor en el eje vertical. 
.container { flex-direction: column; }
 Cambia la dirección de los elementos flexibles para que se dispongan en una columna. 
.item { margin: 10px; }
 Establece un margen de 10 píxeles alrededor de cada elemento flexible. 
.container { flex-wrap: wrap; }
 Permite que los elementos flexibles se envuelvan en la siguiente línea si no hay suficiente espacio.
