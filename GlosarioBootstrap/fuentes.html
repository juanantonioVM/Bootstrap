Fuentes en CSS

La propiedad @font-face permite incorporar fuentes externas, ampliando las opciones tipográficas.

Ejemplo Descripción 
@font-face {
    font-family: 'MiNuevaFuente';
    src: url('fuentes/MiNuevaFuente.ttf');
}
Declara una nueva fuente llamada MiNuevaFuente usando un archivo TTF. 
h1 {
    font-family: 'MiNuevaFuente', Arial, sans-serif;
}
Aplica la fuente personalizada MiNuevaFuente al encabezado h1, con alternativas si no se carga. 
font-style: italic;
Define la fuente en cursiva, aplicable a cualquier elemento de texto. 
font-weight: bold;
Aplica negrita al texto, dándole mayor énfasis. 
body { font-family: 'Arial', sans-serif; }
 Establece la fuente Arial para el texto del cuerpo del documento. 
h1 { font-size: 24px; }
 Define un tamaño de fuente de 24 píxeles para todos los encabezados <h1>. 
p { font-weight: bold; }
 Aplica un peso de fuente en negrita a todos los párrafos. 
h2 { font-style: italic; }
 Aplica un estilo de fuente en cursiva a todos los encabezados <h2>. 
div { text-transform: uppercase; }
 Transforma el texto de los div a mayúsculas.
