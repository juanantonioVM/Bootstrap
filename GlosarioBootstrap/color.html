Propiedades de Color en CSS

Ejemplo Descripción 
p { color: red; }
 Aplica el color rojo al texto de todos los párrafos. 
h1 { color: #00ff00; }
 Aplica el color verde (hexadecimal) al texto de todos los encabezados <h1>. 
div { background-color: rgba(0, 0, 255, 0.5); }
 Aplica un fondo azul semitransparente a todos los div. 
a { color: rgb(255, 165, 0); }
 Aplica el color naranja (RGB) a todos los enlaces. 
body { color: #fff; }
 Aplica color blanco al texto de todo el cuerpo del documento. 
button { background-color: #000; color: #fff; }
 Aplica un fondo negro y texto blanco a todos los botones. 
li { color: hsl(120, 100%, 50%); }
 Aplica un color verde (HSL) a todos los elementos de lista.
